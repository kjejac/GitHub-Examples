## Code

### Inline code

You can print to the terminal using the `puts "hello world"` command.

### Codeblock / Multi line code

#### Without highlighting 

```
def hello_world
    puts "Hello World"
end
```

#### With highlighting

```rb
def hello_world
    puts "Hello World"
end
```
